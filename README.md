# Questionnare

端末内だけでアンケートの作成・回答・集計ができる、SwiftUI製のiOSアプリです。
アンケートはQRコードでほかの端末へ共有でき、共有時にもインターネット接続を必要としません。

## 主な機能

- アンケートの新規作成・編集・削除
- 6種類の質問形式
  - 短文
  - 長文
  - 単一選択
  - 複数選択
  - 5段階評価
  - はい／いいえ
- 必須回答と選択肢の設定
- 質問の並べ替え
- 満足度調査、イベント感想、出欠確認、社内リクエストのテンプレート
- 回答の保存・修正・削除
- 回答結果の自動集計と個別回答の確認
- 回答結果のCSV書き出し（Excel対応）
- QRコードによるアンケート共有・読み取り
- パスワード保護された管理者設定と削除権限の制御

## データとプライバシー

アンケートと回答は、端末のApplication Support領域へJSON形式で保存されます。
外部サーバーへの送信は行いません。

QRコードには質問内容のみが含まれ、回答データは共有されません。QRコードの生成・読み取りもオフラインで動作します。

## 動作環境

- Xcode 26.5以降
- iOS 26.5以降
- Swift 5

プロジェクトの現在のDeployment TargetはiOS 26.5です。

## セットアップ

1. このリポジトリをクローンします。

   ```bash
   git clone https://github.com/Jan12-13/Questionnare.git
   ```

2. `questionnare.xcodeproj` をXcodeで開きます。
3. `questionnare` スキームを選択し、Simulatorまたは実機で実行します。

QRコードの読み取りにはカメラを使用します。実機で初回利用時にカメラアクセスを許可してください。

## 技術構成

- SwiftUI
- Combine
- AVFoundation
- Core Image
- JSONによるローカル永続化

外部ライブラリは使用していません。
